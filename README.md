# AstrBot Essential

为 AstrBot 支持基本的早晚安记录、王者战力查询、城际路线查询、AI绘画和点歌功能。

支持功能：
- 早安/晚安记录
- 王者战力查询
- 某际路线查询
- AI绘画
- 点歌

安装方法：
```
plugin i d-g-n-c-j
```

使用方法：
- `早安`：在群里说早安，记录早安时间
  - 会计算昨晚的睡眠时长
  - 有30分钟CD，防止频繁发送
- `晚安`：在群里说晚安，记录晚安时间
  - 显示你是群里今天第几个睡觉的
  - 有30分钟CD，防止频繁发送
- `战力查询 <英雄名>`：查询王者荣耀英雄的战力信息
  - 示例：`战力查询 小乔`
  - 示例：`战力查询 孙悟空`
- `路线查询 <出发地> <目的地>`：查询城际路线信息
  - 示例：`路线查询 广州 深圳`
  - 示例：`路线查询 北京 上海`
- `ai绘画 <提示词>`：根据提示词生成AI绘画
  - 示例：`ai绘画 一条狗`
  - 示例：`ai绘画 美丽的风景`
- `点歌 <歌名>`：搜索并播放音乐
  - 示例：`点歌 泡沫`
  - 示例：`点歌 唯一`

功能说明：
- **早安/晚安记录**：记录用户的睡眠时间，培养良好作息习惯
  - 早安时会显示昨晚睡了多久
  - 晚安时会显示你是群里今天第几个睡觉的
  - 数据持久化保存，重启不丢失
  - 30分钟CD机制，防止频繁发送
  - 支持多群聊环境
  - 异步处理，不会阻塞其他功能
  - 详细的睡眠时长统计

- **王者战力查询**：查询王者荣耀英雄的战力信息
  - 支持查询所有英雄的战力数据
  - 默认查询安卓-QQ区的数据
  - 显示国服、省标、市标、区标的最低战力要求
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **城际路线查询**：查询城市间的出行路线信息
  - 支持查询任意城市间的路线
  - 显示总距离、总耗时、油费、过桥费、总费用
  - 提供详细的路况信息
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **AI绘画**：根据提示词生成精美的AI绘画
  - 支持中英文双语提示词
  - 默认生成1024x1024高清图片
  - 使用最高guidance值，生成与提示词高度匹配的图片
  - 每次生成1张图片
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **点歌**：搜索并播放音乐
  - 支持网易云音乐搜索
  - 搜索后显示歌曲列表供用户选择
  - 30秒超时机制，超时后提示重新点歌
  - 支持语音模式播放音乐
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制
