# AstrBot Essential

为 AstrBot 支持基本的早晚安记录、王者战力查询和城际路线查询功能。

支持功能：
- 早安/晚安记录
- 王者战力查询
- 城际路线查询
- AI绘画
- Minecraft服务器查询
- 代理IP获取
- 油价查询
- QQ估价
- 星座运势

安装方法：
```
plugin i d-g-n-c-j
```

使用方法：
- `早安`：在群里说早安，记录早安时间
  - 会计算昨晚的睡眠时长
  - 有30分钟CD，防止频繁发送
- `晚安`：在群里说晚安，记录晚安时间
  - 显示你是群里今天第几个睡觉的
  - 有30分钟CD，防止频繁发送
- `战力查询 <英雄名>`：查询王者荣耀英雄的战力信息
  - 示例：`战力查询 小乔`
  - 示例：`战力查询 孙悟空`
- `路线查询 <出发地> <目的地>`：查询城际路线信息
  - 示例：`路线查询 广州 深圳`
  - 示例：`路线查询 北京 上海`
- `绘画 <提示词>`：根据提示词生成AI绘画
  - 示例：`绘画 一条狗`
  - 示例：`绘画 美丽的风景`
- `mc服务器 <服务器地址>`：查询Minecraft服务器状态
  - 示例：`mc服务器 2b2t.org`
  - 示例：`mc服务器 hypixel.net`
- `代理ip`：获取最新的socks5代理IP信息
  - 示例：`代理ip`
- `油价查询 <城市名>`：查询指定城市的油价信息
  - 示例：`油价查询 上海`
  - 示例：`油价查询 北京`
- `qq估价 <QQ号>`：查询指定QQ号的估价信息
  - 示例：`qq估价 123456`
  - 示例：`qq估价 298582245`
- `星座运势 <星座名>`：查询指定星座的运势图片
  - 示例：`星座运势 白羊`
  - 示例：`星座运势 白羊座`
  - 示例：`星座运势 处女`
  - 示例：`星座运势 处女座`

功能说明：
- **早安/晚安记录**：记录用户的睡眠时间，培养良好作息习惯
  - 早安时会显示昨晚睡了多久
  - 晚安时会显示你是群里今天第几个睡觉的
  - 数据持久化保存，重启不丢失
  - 30分钟CD机制，防止频繁发送
  - 支持多群聊环境
  - 异步处理，不会阻塞其他功能
  - 详细的睡眠时长统计

- **王者战力查询**：查询王者荣耀英雄的战力信息
  - 支持查询所有英雄的战力数据
  - 默认查询安卓-QQ区的数据
  - 显示国服、省标、市标、区标的最低战力要求
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **城际路线查询**：查询城市间的出行路线信息
  - 支持查询任意城市间的路线
  - 显示总距离、总耗时、油费、过桥费、总费用
  - 提供详细的路况信息
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **AI绘画**：根据提示词生成精美的AI绘画
  - 支持中英文双语提示词
  - 默认生成1024x1024高清图片
  - 使用最高guidance值，生成与提示词高度匹配的图片
  - 每次生成1张图片
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **Minecraft服务器查询**：查询Minecraft服务器状态
  - 支持查询任意Minecraft服务器的状态
  - 显示服务器是否在线、IP地址、端口、玩家数、最大玩家数和版本信息
  - 支持域名或IP:端口格式的服务器地址
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **代理IP获取**：获取最新的socks5代理IP信息
  - 默认获取socks5协议的代理IP
  - 显示获取时间、代理类型和完整的代理IP地址
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **油价查询**：查询指定城市的油价信息
  - 支持查询全国各城市的油价
  - 显示92号、95号、98号汽油和0号柴油的价格
  - 提供油价趋势预测
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **QQ估价**：查询指定QQ号的估价信息
  - 支持查询任意QQ号的估价
  - 显示QQ号的评估价格
  - 提供QQ号的特点和位数信息
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制

- **星座运势**：查询指定星座的运势图片
  - 支持查询所有星座的运势
  - 直接返回运势图片，包含贵人方位、贵人星座、幸运数字、幸运颜色、爱情运势等信息
  - 支持简写和全名（如"白羊"和"白羊座"）
  - 异步请求，不会阻塞其他功能
  - 详细的错误处理机制
